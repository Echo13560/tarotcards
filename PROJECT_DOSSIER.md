## 🌟 二、核心功能全景图

### ① 占卜系统（核心）

**6 种专业塔罗牌阵**
- 🎴 单牌指引（1 张） — 快速答案
- 🔮 三牌时光（3 张） — 过去/现在/未来
- 💫 爱情牌阵（3 张） — 你/TA/关系走向
- ⚖️ 决策牌阵（3 张） — A/B/建议
- ✦ 十字牌阵（5 张） — 深度全局分析
- 🔯 凯尔特十字（10 张） — 终极人生议题牌阵

**完整的塔罗解读引擎**
- 78 张韦特塔罗牌完整数据（中文 + 英文 + 正逆位 + 5 大主题含义 + 关键词）
- 自动识别问题领域（感情/事业/财运/健康/心灵）
- 综合多张牌生成「逐张解读 + 整体能量分析 + 行动建议」
- 元素（火水风土）能量统计、大小牌占比、逆位率分析

**仪式感四步流程**
1. **写问题** → 静心冥想 (Step 1)
2. **能量校准** → 3.6 秒默念三遍 + 三圈金色脉冲光环
3. **32 秒呼吸冥想** → 跟随光球吸气/屏息/呼气/休息
4. **洗牌仪式** → 8 张牌扇形展开打散 → 重新堆叠
5. **切牌仪式** → 手指上下滑动 30px → 牌堆"咔"地分开
6. **扇形选牌** → 14-20 张卡按 70° 弧度展开 → 选中向上飞出
7. **逐张翻牌** → 解读结果

### ② AI 深度解读（差异化）

**接入硅基流动等免费大模型**
- 默认推荐硅基流动（注册送 14 元额度 ≈ 数千次免费解读）
- 支持 DeepSeek / OpenAI / 任何 OpenAI 兼容协议
- API Key 本地混淆存储（btoa + reverse）

**4 种解读人格**
- 🌸 温柔姐姐 — 共情陪伴
- 💅 毒舌闺蜜 — 直白戳破
- 🧭 理性顾问 — 冷静分析
- 🌙 神秘巫女 — 玄妙古典

**多轮 AI 对话追问**
- 解读结果页底部固定追问输入框
- 3 个建议追问 chip（还有希望吗？/需要注意什么？/行动建议）
- 流式输出 + 用户/AI 气泡 UI
- 对话历史持久化（再次查看历史时能看到完整对话）
- AI 不可用时自动降级到本地模板

**澄清牌（Clarifier Card）**
- 每张牌右上角"✦ 澄清"按钮
- 一键再抽一张针对该牌的深入牌
- 紫光左边框样式，与原牌阵视觉分离

**24 小时解读缓存**
- 同一组牌+同一问题+同一人格 → 本地命中
- 大幅省 token

### ③ 每日塔罗

- **今日之牌**：每天一张固定（日期种子哈希，保证 24h 内不变）
- **🆕 今日宜忌**：基于牌的元素 + 正逆位 + 关键词生成「3 宜 + 2 忌」
- **连续打卡 streak**：未抽 → 宠物心情 -1；连续打卡解锁徽章
- **PWA 应用徽章**：未翻牌显示 1 红点，已翻显示 streak 数字
- **本地通知**：每天定时推送（可设时间）

### ④ 学习板块

- **78 张牌完整百科**：每张牌有 emoji/英文名/关键词/正逆位/5 主题含义
- **搜索 + 5 分类标签**：大牌/权杖/圣杯/宝剑/星币
- **🆕 今日学习一牌**：每天推荐 1 张 + 22 张大牌专属记忆口诀
- **🆕 学习打卡**：连续学习天数 streak，配合徽章

### ⑤ 个性化系统

**生日数字学**
- 本命人格牌（Personality Card）
- 本命灵魂牌（Soul Card）
- 命运数字（Life Path Number 1-9 + 大师数 11/22/33）

**🆕 流年牌（Personal Year Card）**

- 22 张大牌各有专属"年度主题+焦点+建议"三段式文案
- 首页紫金横幅展示 + 距下个生日倒计时

**Onboarding 7 步心理画像**
1. 欢迎 → 名字 → 生日
2. 关注主题（感情/事业/财运/心灵）
3. 解读语气（4 种人格）

**12 星座 + 12 生肖伴侣**
- 自动根据生日识别
- 3D 立体卡通形象（替代 emoji）

**🆕 5 套主题氛围**

- 🌅 黎明微光
- 🌿 翡翠秘境
- ☀ 日光羊皮（白天浅色模式）
- **🆕 自动跟随系统配色**

### ⑥ 数据沉淀

**塔罗日记系统**
- 写感悟、关联牌阵
- **🆕 AI 周复盘**：每周自动生成「3 个最强信号 + 1 段核心洞察」

## 📜 八、技术决策回顾

### 为什么是 PWA 而不是原生 App？
- **跨平台一份代码** —— iOS/Android/Mac/Windows 全覆盖
- **无需上架审核** —— 改完立刻发布
- **完整离线** —— Service Worker 缓存全部资源
- **零安装门槛** —— 不需要 App Store / Google Play
- **后续可包装** —— 用 Capacitor 可一键打包成原生 App

### 为什么是纯静态零后端？
- **零运维成本** —— GitHub Pages 永久免费
- **隐私友好** —— 用户数据全在本地，不上传服务器
- **极快加载** —— 没有 API 等待，进入即用
- **可移植** —— 复制文件夹到任何静态托管即可部署

### 为什么自己实现塔罗解读引擎而不是只用 AI？
- **AI 不可用时也能用** —— 本地解读引擎做兜底
- **24h 缓存** —— 同问题不重复消耗 token
- **可控性** —— 不靠 AI 也能保证基本体验
- **响应速度** —— 本地解读零等待

### 为什么用 Web Audio API 程序化音效？
- **零依赖** —— 不需要音频文件
- **可控性** —— 音量/音色完全可调
- **离线可用** —— 不依赖网络资源
- **包体小** —— 整个 App 不到 5MB
